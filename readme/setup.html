Setting Up Freya

Windows:

1. Download a .zip archive from http://freya.dns.st/forum/index.php?act=idx. I'll assume you're using the text version if you need to read this guide. Unzip the files to the C: (or whatever your main HDD is) drive.

2. Carefully go through each of the .conf files and edit them to suit your needs. If you don't understand something, you can get help on Freya's forums.

3. Add a user using adduser.exe. Add your user ID (can be found in freya/save/account.txt) to the GM_accounts.txt, found in freya/conf/, for use as a Game Master. In the future you can make accounts at login by prefixing the username with M_ (male) or F_ (female), if enabled.

4. Start the server by running login-server.exe, char-server.exe, and map-server.exe.

5. Enjoy your server! If you have any problems, you can receive help at the forums.


Linux:

1. Download a .tar.gz or .tar.bz2 archive from http://freya.dns.st/forum/index.php?act=idx. I'll assume you're using the source version because it's the most proper way of going about it. Untar the files to your $HOME directory.

2. Go into a terminal and run "./configure --help" in your freya directory. Look through the options and decide which ones you're going to use. Now run "./configure [options you're going to use]" followed by "make" if successful.

Updating Freya

1. Install Freya in a new directory.

2. Transfer your old /save/ folder contents (Athena.txt, Account.txt, Storage.txt, etc.) to the new /save/ folder.

3. Transfer/Edit grf-files.txt, gm-account.txt and motd.txt as neccessary.

4. Transfer any extra custom NPCs to the /npc/ folder and edit if neccessary.

5. Edit .conf files to setup your server.

6. You're done!


NOTE: It is not recommended to transfer your DB files as the DB files are constantly updated by the Freya/jAthena team. Just readd your changes to the new DB files instead of replacing the new DB file with your old one.
